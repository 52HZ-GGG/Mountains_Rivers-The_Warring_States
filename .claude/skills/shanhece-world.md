# 《山河策》核心游戏世界知识

## 游戏类型与目标
- 类型：回合制历史策略游戏（4X：探索、扩张、开发、征服）
- 平台：PC (Steam)，可移植移动端
- 团队：1 程序 + 1 美术 + 1 策划
- 数据驱动：所有数值、兵种、事件均存放在 `data/*.json`，禁止硬编码

## 核心玩法循环
每回合：内政经营 → 科技文化 → 民心管理 → 外交交涉 → 军事行动 → 回合结算

## 七大核心系统要点
### 1. 战争系统（重在强地理制约）
- 移动力消耗：平原 1，森林 2，山地 3，沼泽 4，河流需渡口/桥梁
- 特殊地形：山脉不可通行（除非栈道科技或隘口），冬季北方河流结冰可通行
- 关隘：天然关隘（地图固定标记，如函谷关、武关）+ 可造箭楼（玩家在山地隘口建造，消耗金钱+铁，对相邻 1 格敌军额外伤害，随关隘陷落失效）；攻击方伤害 -50%，防守方 +50%，需攻破才可进入后方
- 战术：伏击（被动，森林/山地概率触发，双倍伤害）、夹击、断粮（BFS 从部队位置向所有友方城市探索可通行路径，无路则断粮，每回合 -5% 粮 -3% 兵）、火攻（主动技能，夏/秋可选；**与伏击互斥，主动选择火攻则跳过伏击判定**）
### 2. 经营系统
- 资源：粮、钱、铁；每城随机特产（产马城骑兵训练速度+30%、**产铁城铁矿产量+30% 且武器升级费用-20%**、盐池城金钱收入+20%）
### 3. 文化传播
- 文化值邻格渗透；敌城文化 <50% 可能倒戈/暴乱
### 4. 科技（三线）
- 军事：架桥/浮桥、栈道、弩炮；内政：郡县制、水利；文化：百家争鸣
### 5. 民心
- 0~100，高民心加税收征兵，低民心生变
### 6. 外交
- 好感度 -100~+100；AI 逻辑：远交近攻、均势反应、可能背信
- 国际声望（隐藏指标，不在 UI 显示）：信守条约 +5，不宣而战 -20，背盟 -30
- **游士**：外交面板触发，消耗金钱+本国文化值，对目标敌方城市文化值 +X，冷却 3 回合
### 7. 百家学派（六家）
- 儒家（民心上限+20，文化胜利阈值60%）、法家（科技+20%，法治胜利 TBD）、
  墨家（防御+25%，和平胜利）、道家（维护费-50%，自然胜利：**我方控制城市辖区内**森林/山脉地块占比>40%，辖区半径默认 3 格）、
  兵家（伏击+50%，军事胜利30%城市）、纵横家（外交+50%，同盟胜利4盟友维持10回合；⚠️ 阶段4平衡测试再调，候选下调至3）
- 切换学派需金钱和回合数（变法），期间民心下降

## 季节系统
- **4 回合 = 1 年**（春→夏→秋→冬→春），每年春季首回合触发「年度事件」判定
- 默认游戏时长：30~50 年（120~200 回合）
- 季节修正：春（粮产+20%，征兵-20%，民心+5）、夏（火攻可用，民心-5）、秋（收获×2，火攻可用，民心+10）、冬（粮耗+30%，行军损耗+50%，北方河结冰，骑兵移动-1，民心-10）
- 季节铭言：半文言短句，如"风萧萧兮易水寒"
- **学派-季节联动**：道家冬季粮食消耗惩罚减半、儒家春秋两季文化传播+10%、兵家夏秋火攻额外伤害+20%

## 将军系统
- **基础属性**：武力（1-100，每点+0.3%近战伤害）、智力（1-100，每点+0.2%伏击/夹击概率）、统率（1-100，每点-0.1%兵力损耗）、经验（0~∞，每100经验升1级，最高5级）
- **招募**：兵营每4回合产出武力型将军、学宫每4回合产出智力型将军、随机事件招揽（如「商鞅入秦」）、战场提拔（无名小卒成长为将军）
- **成长**：每次参战获10~30经验，每升1级解锁1个特技槽位（擅伏/铁壁/神速/识破/连环等）
- **委派**：每支部队最多1名将军，委派/调动消耗1回合
- **阵亡**：部队全歼时 50% 战死、30% 被俘、20% 逃归；被俘将军可被对方说降使用
- **学派/国家联动**：兵家经验获取+50%、秦国新将武力偏高（+10）、赵国骑兵特技解锁+30%、魏国步兵无将军也+10%经验

## 国家特色（七雄）
- 秦：锐士（重步+20%攻防）、尚武氛围
- 赵：胡服骑兵（轻骑移+1，森林正常移）、骑兵经验+30%
- 齐：技击手（远程弩射程+1）、文化传播+20%
- 楚：申息之师（步+弓，山地无视惩罚）、随机正面事件
- 魏：武卒（重步击杀回血）、步兵自动升一级
- 燕：辽东弓骑（骑射移动后射击）、被宣战战力+30%
- 韩：劲弩（攻城+50%，无法近战）、武器升级费用-40%

## 文本风格
- 事件与叙述采用半文言，如"天旱不雨，仓廪渐空。"
- 攻破都城全屏公告："某都破，某国宗庙隳。"

# 山河策

**寸土必争，变法图强 —— 一款轻量级战国回合制策略游戏**

## 游戏简介

《山河策》是一款以战国时期为背景的回合制历史策略游戏（4X类），玩家扮演一国之主，在**战争、内政、文化、科技、民心、地理、外交**七重维度中博弈，最终称霸中原。

游戏突出百家争鸣、合纵连横、强干弱枝、变法图强的战国精神，核心特色在于**"地理即命运"**——山川险阻直接决定战争形态，寸土必争。

## 核心玩法

每回合玩家依次执行：

1. **内政经营** — 分配资源（粮、钱、铁），建造/升级建筑
2. **科技与文化** — 投资科技树、选择学派、推行文化政策
3. **民心管理** — 处理民生事件、调整税率、征兵政策
4. **外交交涉** — 缔结条约、宣战、求和、借道、索要通关权
5. **军事行动** — 调兵遣将、攻城略地，严格受地形与天堑制约
6. **回合结算** — 触发随机事件、AI同步行动、季节更替

## 七大核心系统

### 战争系统 & 天堑系统

- 五行兵种相克：步兵、弓兵、骑兵、战车、攻城器械
- 天堑地形：山脉不可通行、大河隔绝两岸、关隘易守难攻、沼泽损兵折将
- 战术机制：伏击、夹击、断粮、火攻

### 经营系统

- 三大核心资源：粮食、金钱、铁矿
- 城市建设：农田、市集、矿场、兵营、城墙、学宫、祠堂、驿站
- 城市特产：每座城随机赋予独特加成

### 文化传播系统

- 文化值向外辐射，影响邻国城市民心
- 敌城文化值低于阈值可触发"倒戈"或"暴乱"
- 文化胜利：覆盖天下70%城市并维持10回合

### 科技发展系统（三线并行）

| 科技线 | 关键节点 |
| :--- | :--- |
| 军事 | 架桥/浮桥、栈道、弩炮、铁甲兵 |
| 内政 | 郡县制、水利工程 |
| 文化 | 百家争鸣、刻石颂德 |

### 民心所向系统

- 数值范围 0~100，影响税收、征兵、暴动概率
- 调控手段：税率调整、赈济、祭祀、水利

### 外交系统（合纵连横）

- 外交动作：宣战、停战、互不侵犯、军事通行权、商路、进贡、结盟
- AI 决策逻辑：远交近攻、均势反应、无信义原则

### 百家学派系统

| 学派 | 核心思想 | 特色 |
| :--- | :--- | :--- |
| 儒家 | 仁政、礼治 | 民心+20，文化传播+30%，文化胜利阈值降至60% |
| 法家 | 法治、耕战 | 科技+20%，征兵+20%，法治胜利 |
| 墨家 | 兼爱、非攻 | 防御+25%，守城器械半价，和平胜利 |
| 道家 | 无为而治 | 维护费-50%，人口+30%，自然胜利 |
| 兵家 | 出奇制胜 | 将军经验+50%，伏击+50%，军事胜利门槛降至30% |
| 纵横家 | 合纵连横 | 外交好感变化+50%，同盟胜利 |

## 开发计划

| 阶段 | 周期 | 目标 |
| :--- | :--- | :--- |

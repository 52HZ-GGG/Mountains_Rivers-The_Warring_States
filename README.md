# 山河策

**寸土必争，变法图强 —— 一款轻量级战国回合制策略游戏**

## 游戏简介

《山河策》是一款以战国时期为背景的回合制历史策略游戏（4X类），玩家扮演一国之主，在**战争、内政、文化、科技、民心、学派、外交**七重维度中博弈，最终称霸中原。

游戏突出百家争鸣、合纵连横、强干弱枝、变法图强的战国精神，核心特色在于**"地理即命运"**——山川险阻直接决定战争形态，寸土必争。

## 核心玩法

每回合玩家依次执行：

1. **内政经营** — 分配 7 种资源（粮/金/木/马/精铁/匠人/建材），建造/升级建筑，管理大夫
2. **科技与文化** — 投资 66 项科技树、选择学派、推行文化政策
3. **民心管理** — 处理民生事件、调整税率（5 档）、征兵政策、关注安定度
4. **外交交涉** — 缔结条约、宣战、停战谈判、商路、附庸、情报力经营
5. **军事行动** — 调兵遣将、攻城略地，严格受地形、天堑、季节制约
6. **回合结算** — 触发随机事件（88 个）、AI 同步行动、季节更替、人口增长

## 核心系统

### 战争系统 & 天堑系统

- 五大兵种类别相克：步兵、骑兵、弓兵、攻城、水军（5×5 分类克制矩阵）
- 19 种基础兵种 + 7 国特色兵种变体
- 天堑地形：山脉不可通行、大河隔绝两岸、关隘（可占领，有 HP）、深海阻隔
- 11 种地形影响攻防修正、伏击概率、移动力消耗
- 战术机制：伏击、夹击、包围、断粮、火攻、远程双模式、ZOC（控制区）

### 经营系统

- 七大资源：粮食、金钱、木材 + 马匹、精铁、工匠、建材（四种特殊资源）
- 50 座城市，5 级体系（村/镇/城/府/都），28 种建筑分 5 类（经济/军事/防御/政治/特殊）
- 建筑辖区系统：以城市为中心 6 格六角环，建筑实体建在格子上
- 10 座全局唯一奇观（含竞赛机制）
- 腐败系统、税收系统、特产加成、季节资源修正

### 文化传播系统

- 文化值向外辐射，影响邻国城市民心
- 敌城文化值低于阈值可触发"倒戈"或"暴乱"
- 文化胜利：覆盖天下70%城市并维持10回合

### 科技发展系统

66 项科技，4 类 × 3 时代（早期/中期/晚期），前置链驱动：

| 科技类别 | 数量 | 关键节点 |
| :--- | :---: | :--- |
| 军事 | 29 | 架桥/浮桥、栈道、弩炮、铁甲兵、ZOC 扩展 |
| 经济 | 15 | 郡县制、水利工程、资源产出加成 |
| 民生 | 12 | 百家争鸣、民心加成、人口增长 |
| 建筑 | 10 | 城防强化、高级建筑解锁 |

### 民心与稳定性系统

双层制：国家级民心（0~100）+ 城市级安定度（0~100）

- 民心影响税收效率、征兵、战斗力；4 档阈值效果（>80/50~80/30~50/<30）
- 安定度影响城市产出、叛乱风险；5 档阈值效果（含叛乱/起义）
- 调控手段：税率调整（5 档）、季节修正、学派加成、建筑加成、奇观效果

### 外交系统（合纵连横）

三层外交模型，自下而上联动：

| 层级 | 核心参数 | 作用 |
| :--- | :--- | :--- |
| 第一层：双边关系层 | 好感度（-100~+100，双向独立） | 决定可用外交动作（8 种 + 俘虏释放 + 质子） |
| 第二层：国家声望层 | 声望（0~100，五档模糊指示） | 解锁贸易/联盟等高阶动作门槛 |
| 第三层：天子权威层 | 朝贡度（0~100，每国独立） | 朝贡→册封→声望联动，不朝贡则声望下降 |

- 18 种外交动作：宣战、停战、同盟、附庸、商路、赠礼、质子、朝贡、册封、关隘开放、游士等
- 情报力系统：三层信息可见性（地理视野 + 外交情报力 + 单位信息）
- AI 决策：四维性格驱动（好战/贪婪/诚信/外交），战争目标选择、停战评估、联合抗敌、附庸脱离


### 百家学派系统

六大学派，3 级递进效果，不绑定胜利条件（所有学派均可追求任意胜利路径）：

| 学派 | 核心思想 | 特色效果 |
| :--- | :--- | :--- |
| 儒家 | 仁政、礼治 | 民心上限+10~+30，民心+1~+2/回合，文化传播率+15%~+50%，安定度+3~+8 |
| 法家 | 法治、耕战 | 科研速度+10%~+30%，征兵消耗-10%~-30%，安定度+5~+12 |
| 墨家 | 兼爱、非攻 | 防御+10%~+35%，攻城器械工匠消耗×0.5~×0.3，建筑维护费-10%~-20% |
| 道家 | 无为而治 | 建筑维护费-15%~-50%，粮耗-5%~-15%，人口增长+10%~+40% |
| 兵家 | 出奇制胜 | 将军经验+20%~+60%，攻击+5%~+15%，伏击+30%~+50% |
| 纵横家 | 合纵连横 | 好感效果+20%~+60%，同盟成本×0.85~×0.50，情报+1~+2 格 |

学派通过**基础修正**（每回合自动）和**政策触发**（消耗资源一次性）双轨制影响各系统。

### 粮食·人口·征兵机制

- 以 1 为基本单位的城市人口系统，每人口产出粮食和金钱、自耗粮食
- 人口增长采用进度条制，受粮食盈余、季节、安定度调节
- 征兵从总人口中抽取（上限 20%），服役比例过高惩罚经济和增长
- 饥荒（城市产粮不足）和国家储粮不足均有严重后果

### 季节系统

- 每 4 回合为一年，春夏秋冬循环
- 季节影响：资源产出修正、民心修正、火攻限制（仅夏秋）、河流冻结（冬季）、文化传播减半（冬季）
- 旱灾系统：春季触发，奇观/粮仓/御史台可减灾

### 大夫系统

三类大夫（文/武/外交）替代旧将军系统，覆盖城市管理、军队统帅、外交使节：

| 类型 | 容量 | 任命位置 | 核心属性 |
| :--- | :---: | :--- | :--- |
| 文大夫 | 5 | 城市 | 理财%（金钱加成）+ 安民%（安定度加成） |
| 武大夫 | 5 | 部队 | 勇武%（攻击加成）+ 韬略%（防御加成） |
| 外交大夫 | 2 | 目标国家 | 辩才%（降低成本）+ 亲和%（增长友好度） |

三档品质（普通 60% / 稀有 30% / 传说 10%），最高 5 级，含历史名人专属技能。

## 当前项目特别提醒

1. 优先复用现有命名风格，不要临时创造新前缀。
2. 当前项目中大量资源路径已写死在代码或数据里，交付前要先确认目标路径。
3. 阶段 0~1 允许占位资源，但必须显式标明“占位”。
4. 如果是替换现有资源，请注明“原路径不变，直接覆盖”。

## 当前项目特别提醒

1. 所有平衡数值必须写入 `data/*.json`，不要要求程序硬编码。
2. 已冻结字段名只许加不许改；若必须调整旧字段，先单独确认。
3. 如果一个系统还没有最小闭环，请先明确“本阶段不做什么”，否则程序实现范围会失控。
4. 如果涉及核心系统，请一并写出希望覆盖的测试点。

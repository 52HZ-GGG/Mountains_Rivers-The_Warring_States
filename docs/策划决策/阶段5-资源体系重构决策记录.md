## 决策 #130：铁矿→木材基础资源替换

**状态**：✅ 已定稿

**背景**：原三大核心资源为粮食、金钱、铁矿。铁矿在战国背景下主要用于武器锻造和建筑，但与精铁（高级锻造资源）语义重叠，且木材作为基础建设资源更具通用性。

**决策**：
- 完全删除 `iron` 基础资源接口（变量、方法、数据字段），非重命名
- 新增 `wood`（木材）作为第三种基础资源
- 木材城市基础产出 = 20/回合（原铁矿为 30）
- 季节修正：春×1.0、夏×1.0、秋×1.0、冬×0.8（与原铁矿相同）
- 特产加成：木材特产城产出×1.3，建造成本×0.8

**影响范围**：game_manager.gd、city_manager.gd、event_manager.gd、所有 data/*.json、所有 UI 文件

## 决策 #131：新增工匠/建材两种特殊资源

**状态**：✅ 已定稿

**背景**：原特殊资源为马匹和精铁两种。兵种招募和建筑升级的资源消耗维度不足，需要新增资源类型增加策略深度。

**决策**：
- 新增 `craftsmen`（工匠）特殊资源：高级攻城/水军单位招募消耗
- 新增 `building_materials`（建材）特殊资源：高级建筑升级和奇观建造消耗
- 精铁（refined_iron）保留不变
- 最终特殊资源体系：马匹、精铁、工匠、建材（共 4 种）

**工匠基础参数**：
- 国家级基础产出：3/回合
- 城市基础产出：2/回合
- 专属建筑：工匠坊（craftsmen_workshop），+3 工匠/级，max 3 级
- 季节修正：春×1.0、夏×1.0、秋×1.2、冬×0.8
- 特产加成：工匠特产城产出×1.3

**建材基础参数**：
- 国家级基础产出：3/回合
- 城市基础产出：2/回合
- 专属建筑：建材坊（building_materials_workshop），+3 建材/级，max 3 级
- 季节修正：春×1.2、夏×1.0、秋×1.0、冬×0.7
- 特产加成：建材特产城产出×1.3，建造成本×0.8

## 决策 #132：兵种招募取消木材消耗

**状态**：✅ 已定稿

**背景**：原所有兵种招募均消耗铁矿（cost_iron）。木材作为基础建设资源，不应被兵种招募大量消耗，否则会导致建设与征兵的资源冲突过于激烈。

**决策**：
- 所有 19 个基础兵种删除 `cost_iron` 字段（完全移除，非设为 0）
- 兵种招募仅消耗：金币（cost_gold）+ 粮食（cost_food）+ 马匹（cost_horse）+ 精铁（cost_refined_iron）+ 工匠（cost_craftsmen）
- 木材仅用于建筑建造和升级

## 决策 #133：工匠消耗分配方案

**状态**：✅ 已定稿

**背景**：需要确定哪些兵种消耗工匠资源。工匠代表高级技术劳动力，应与需要复杂工艺的兵种关联。

**决策**：

| 兵种 | 工匠消耗 | 理由 |
|:---|:---:|:---|
| 投石车 (siege) | 5 | 大型攻城器械，需工匠组装 |
| 弩炮 (ballista) | 12 | 精密远程攻城武器，工匠需求最高 |
| 大翼 (great_wing) | 5 | 大型战船，需工匠建造维护 |
| 楼船 (tower_ship) | 10 | 最大水军单位，工匠需求高 |
| 冲车 (battering_ram) | 0 | 简单攻城器械，不需工匠 |
| 蒙冲 (mengchong) | 0 | 轻型战船，不需工匠 |
| 其他兵种 | 0 | 普通兵种不消耗工匠 |

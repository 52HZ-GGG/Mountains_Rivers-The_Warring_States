### 1.2 大夫系统与战斗的接口

**来源**：游戏机制总览 §13

武大夫属性直接接入战斗公式：

```
army_attack  *= (1 + 勇武% / 100)
army_defense *= (1 + 韬略% / 100)
```

替代现有代码中硬编码的 `atk_general_mod = 1.0` 和 `def_general_mod = 1.0`。

大夫三档品质（普通 5~30% / 稀有 15~60% / 传说 30~100%），最高 5 级，每级两个属性各 +5%。武大夫在部队被歼灭时有命运判定（30% 死亡 / 40% 被俘 / 30% 逃脱）。

## 二、未解决（设计缺失，程序无法实现）

以下问题在现有文档中找不到定义，程序需要策划补充后才能实现。

### 2.1 【阻塞】战斗状态机 — 战斗流程未定义

**问题**：文档只说了"回合制战斗"，但没有定义战斗的完整流程。

**需要定义的内容**：

1. **战斗在哪里发生？**
   - 在大地图上直接交锋？
   - 还是进入独立的战术战斗场景（类似演武）？
   - 还是两种模式并存（小战斗在大地图解决，大战役进入战术场景）？

2. **参战方行动顺序？**
   - 速度快的先行动？
   - 攻方先手 / 防方先手？
   - 还是同时结算？

3. **战斗何时结束？**
   - 一方全灭？
   - 一方撤退？
   - 回合数限制？
   - 占领城市格？

4. **多支部队参战规则？**
   - 一个格子能放几支部队？
   - 同一方多支部队是各自独立行动还是合并为一队？
   - 支援范围？（相邻格的友军能否加入战斗？）

**影响范围**：战斗系统整体架构，所有战斗相关代码的前置依赖。

### 2.2 【重要】撤退/溃退机制 ✅ 已解决

**问题**：士气崩溃（< 50）有 -20% HP 和属性减半效果，但没有定义撤退规则。

**需要定义的内容**：

1. 玩家能不能主动命令部队撤退？
2. 崩溃后是自动溃逃（失去控制）还是只是承受 debuff？
3. 撤退时能不能被追击？有没有"追击伤害"？
4. 撤退的目标是最近的友方城市还是任意方向？
5. 撤退消耗移动力吗？能撤多远？

**解决方案**：决策 #162（阶段5-平衡性调整文档.md）、战斗系统.md §6.8
- 主动撤退：玩家可指令，消耗全部行动，自动远离敌军，持续撤退至安全距离 ≥ 3
- 被动溃退：士气 < 20 必定触发，AI 接管，自动移向最近友方城市
- 追击机制：步兵/骑兵自动追击离开 ZoC 的单位，伤害 = 攻击力 × 0.3，无视防御
- 溃退到达友方城市后士气恢复到 30

### 2.3 【重要】区域控制（Zone of Control） ✅ 已解决

**问题**：文档完全没有提及 ZoC 机制。

**需要决定的内容**：

1. 敌军相邻时，我方部队能不能自由移动离开？
2. 如果有 ZoC，是所有单位都生效还是只有步兵/骑兵？
3. 弓兵/攻城器械是否受 ZoC 约束？
4. 离开 ZoC 是否消耗额外移动力？

**设计影响**：没有 ZoC 的话，骑兵可以直接绕过前线打后方城市/弓兵，战术深度大打折扣。

**解决方案**：决策 #161（阶段5-平衡性调整文档.md）、战斗系统.md §5.7/§9.4
- 步兵类和骑兵类对相邻格施加 ZoC（弓兵、攻城、水军不施加）
- 所有单位受约束，斥候类（recon 标签）免疫
- 效果：移动力 ×0.5（独立乘算），离开 ZoC 格额外消耗 1 MP
- 撤退/溃退中的单位不受 ZoC 约束

### 2.4 【重要】夹击判定规则

**问题**：文档定义了夹击 debuff -0.3，但没有定义"不同方向"在六角格上怎么判定。

**需要定义的内容**：

1. 六角格有 6 个邻居，哪些组合算"不同方向"？（相邻的两个邻居算同方向还是不同方向？）
2. 3 支部队从 3 个方向夹击，debuff 是 -0.6（叠加）还是有递减？
3. 夹击是在攻击时实时计算，还是预先标记在单位状态上？
4. 夹击对攻方有加成吗？还是只对防方有 debuff？

### 2.5 【建议】伤害溢出规则

**问题**：当一个单位被打死时，多余伤害怎么处理？

**需要决定**：

1. 多余伤害直接消失？
2. 溢出到相邻友军？
3. 如果溢出，是优先溢出到同格友军还是相邻格？

**设计影响**：这会影响集火策略——如果伤害不溢出，玩家可以放心集火；如果溢出，需要考虑分散攻击。

### 2.6 【建议】远程攻击遮挡与近战惩罚

**问题**：弓兵/弩兵有射程（range: 2~3），但缺少以下规则：

**需要定义**：

1. **遮挡**：中间格有友军/敌军时，远程单位能否越过他们攻击后方目标？
2. **近战惩罚**：远程单位被近身（距离 1）时，攻击力是否下降？下降多少？
3. **高处优势**：山地/城墙上的远程单位是否有射程加成？

### 2.7 【建议】部队战斗后恢复

**问题**：城市有"未受攻击时每回合恢复 5% HP"的规则，但部队单位没有回血机制。

**需要定义**：

1. 部队 HP 能否自然恢复？
2. 在城内/友方城市格时能否回血？回多少？
3. 有没有医疗类建筑/大夫技能提供回血？
4. 如果部队不能回血，是否需要某种"补员"机制（消耗粮食/金钱恢复兵力）？

## 二、命名规范

所有文件遵循 pre-push 正则 `^(tile|unit|ui)_[a-z]+(_[a-z0-9]+)*\.png$`，命名格式：

```
tile_city_{faction_id}_{city_id}.png          ← 非首都
tile_city_{faction_id}_capital.png            ← 首都
```

## 六、程序接入建议

### 加载方式

```gdscript
# 根据 city_id 获取图标路径
func get_city_icon(city_id: String, faction_id: String, is_capital: bool) -> String:
    if is_capital:
        return "res://photos/city/tile_city_%s_capital.png" % faction_id
    return "res://photos/city/tile_city_%s_%s.png" % [faction_id, city_id]
```

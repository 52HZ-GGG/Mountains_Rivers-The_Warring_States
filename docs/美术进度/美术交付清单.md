## 六、程序接入说明

### 程序需要做的

| 事项 | 说明 |
|:---|:---|
| TileMap 集成 | 将地形图块配置为 TileSet，六角网格渲染 |
| 文化覆盖层 | 读取 `factions.json` 颜色，注入 `culture_overlay.gdshader` 的 `faction_color` |
| UI 布局 | 将面板/按钮/图标在场景中布局，绑定交互逻辑 |
| 动画播放 | 用 SpriteFrames 或 AnimationPlayer 播放兵种动画 |
| 字体导入 | 导入 `pixel_lishu.fnt`，设为 Label 的 font |
| Shader 绑定 | 用 `shader_helpers.gd` 创建 material 并赋给对应节点 |

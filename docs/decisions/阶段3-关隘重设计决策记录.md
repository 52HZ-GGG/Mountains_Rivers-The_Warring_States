## 关键决策

### 决策 84：关隘 HP 系统 — 可占领地形 + 城防血量

**选定方案**：
- 关隘从天堑屏障改为可占领特殊地形，拥有独立 HP
- 关隘 HP = 500（`fortification.pass_hp`），城市 HP = 1000（`fortification.city_hp`）
- 攻城器械对关隘/城市造成 3 倍伤害（`siege_damage_multiplier = 3.0`）
- 普通单位对关隘/城市造成正常伤害

**理由**：
- HP 系统让关隘成为有战略价值的争夺目标，而非单纯的行军障碍
- 攻城器械克制关隘/城市，强化兵种分工（攻城器械从"可选"变为"必要"）
- 关隘 HP 500 低于城市 HP 1000，体现关隘防御弱于城市的历史逻辑
- 与 fortification 系统统一，共用 `hp_type: "fortification"` 标识

**已落地**：
- `data/terrain.json` → pass 条目：`is_occupiable: true`, `structure_hp: 500`, `hp_type: "fortification"`
- `data/balance_params.json` → `fortification` 段

### 决策 85：关隘易主机制 — HP 归零 + 驻军清除

**选定方案**：
- 关隘 HP 降至 0 且驻守单位被全歼时，敌方单位移动到该关隘格子上即完成易主
- 易主后关隘 HP 重置为满（500），归属权变更为占领方
- 未满足条件（HP > 0 或仍有驻军）时，敌方单位无法占领

**理由**：
- 双重条件（HP + 驻军）防止"一回合速占"，要求攻方投入足够兵力
- HP 重置让新占领方立即获得防御能力，避免"空城"状态
- 与城市占领机制逻辑一致（城市也是 HP 归零 + 驻军清除后易主）

**已落地**：`docs/数值公式.md` §1.5 关隘战斗公式

### 决策 86：关隘行军降速 + 选择性文化传播阻断

**选定方案**：

**行军降速**：
- 关隘不阻挡行军，但通过关隘格子时移动速度 ×0.5（`crossing_rules.move_speed_modifier = 0.5`）
- 关隘战斗中攻方攻击修正 -0.3（`atk_mod: -0.3`），守方防御加成 +0.5（`def_bonus: 0.5`）

**文化传播阻断**：
- 关隘阻断非占领国的文化传播（`blocks_culture_except_owner: true`）
- 占领国的文化可正常通过关隘向外扩散
- 非占领国的文化无法穿越关隘格子

**理由**：
- 降速效果让关隘成为"减速带"而非"路障"，保持行军流畅性的同时增加战术考量
- 攻守修正强化关隘的防御价值（守方 +0.5 防御，攻方 -0.3 攻击）
- 选择性文化传播阻断让关隘成为文化战的关键节点：占领关隘 = 控制文化扩散方向
- 与游戏机制总览§3 文化系统一致："关隘阻断非占领国文化传播"

**已落地**：
- `data/terrain.json` → pass 条目：`crossing_rules.move_speed_modifier`, `crossing_rules.blocks_culture_except_owner`
- `docs/数值公式.md` §4.7 边界条件

### 决策 87：城市 HP 系统 + 人工关隘建造预留

**选定方案**：

**城市 HP**：
- 城市 HP = 1000（`fortification.city_hp`），与关隘共用 fortification 系统
- 攻城器械对城市同样造成 3 倍伤害
- 城市 HP 归零且驻军清除后，城市易主（逻辑与关隘一致）

**人工关隘建造**：
- 预留接口 `artificial_pass_enabled: false`，当前不实现具体内容
- 后续阶段可在特定地形（如山口、渡口）建造人工关隘
- 人工关隘继承天然关隘的 HP、易主、文化传播阻断规则

**理由**：
- 城市 HP 系统与关隘 HP 系统统一设计，降低实现复杂度
- 攻城器械同时克制城市和关隘，强化兵种协同（步兵/骑兵保护攻城器械）
- 人工关隘建造预留接口，为后续阶段扩展提供设计空间（如"函谷关"类历史名关）
- `artificial_pass_enabled: false` 明确当前阶段不实现，避免范围蔓延

**已落地**：
- `data/balance_params.json` → `fortification.city_hp`, `fortification.artificial_pass_enabled`
- `docs/游戏机制总览.md` §11 地理与天堑（人工关隘建造预留说明）

# 阶段2增补：兵种扩展与特殊资源决策记录

## 概述

本次增补扩展了兵种体系（从12个扩展至19个）、引入两种特殊资源（马匹、精铁）、新增视野机制（vision_range）、设计士气参数、新增9个科技节点、1个奇观，并修复了若干数据不一致问题。

---

### 1.1 改名

| `navy` | 水军 | `mengchong` | 蒙冲 | ID同步修改 |

### 1.4 骑兵三拆分

- **护卫骑兵**（原骑兵）：改名，定位为基础骑兵，消耗少量马匹
- **重骑兵**：高攻高防，低速，高补给消耗，需马匹+精铁
- **突击骑兵**：极速（speed=7），`momentum_attack`特技，仅需马匹

### 1.5 水军体系

- **蒙冲**（原水军）：基础水军单位，开局需科技解锁
- **大翼**：主力战舰，中期科技解锁
- **楼船**：水军旗舰，晚期科技解锁，属性最高

## 2. 视野机制（vision_range）

为所有兵种新增 `vision_range` 字段，为战争迷雾系统做数据准备。

| 兵种类型 | vision_range |
|----------|-------------|
| 大部分陆地单位 | 2 |
| 蒙冲/楼船/大翼 | 3（水上视野开阔） |
| 弓骑兵 | 3 |
| 斥候小队 | 4 |
| 斥候骑兵 | 5 |

- 视野范围以格为单位，影响单位周围的战争迷雾揭示
- `recon` 特殊标识的单位视野更大
- 战争迷雾的实际渲染逻辑在阶段3实现

## 3. 士气机制

在 `balance_params.json` 新增 `unit_morale` 节，定义单位级士气参数：

| 参数 | 值 | 说明 |
|------|-----|------|
| `base_morale` | 100 | 初始士气 |
| `morale_break_threshold` | 20 | 士气崩溃阈值 |
| `retreat_chance_at_break` | 0.5 | 崩溃时溃逃概率 |
| `morale_loss_on_defeat` | 15 | 战败士气损失 |
| `morale_loss_on_ally_death` | 5 | 友军阵亡士气损失 |
| `morale_recovery_per_turn` | 3 | 每回合恢复 |
| `morale_recovery_in_city` | 8 | 城中每回合恢复 |
| `low_morale_atk_penalty` | 0.3 | 低士气攻击惩罚 |
| `high_morale_atk_bonus` | 0.1 | 高士气攻击加成 |

- 每个单位实例有独立士气值（运行时状态，由战斗系统管理）
- 士气跌破阈值时有概率溃逃
- 战斗系统中的士气实现在阶段3完成

## 4. 特殊资源系统

### 4.1 资源类型

| 资源 | 基础产出 | 城市自带产出 | 说明 |
|------|----------|-------------|------|
| 马匹（horse） | 5/回合 | 2/回合 | 通过马厩建筑或特定城市获得 |
| 精铁（refined_iron） | 3/回合 | 2/回合 | 通过精铁坊建筑或特定城市获得 |

### 4.2 获取机制

**建筑产出**（受`max_national_count`限制）：
- 马厩（`stable`）：仅限`special_resource == "horse"`的城市建造，全国最多3座
- 精铁坊（`ironworks`）：仅限`special_resource == "iron"`的城市建造，全国最多3座

## 8. 兵种分类克制系统

### 8.1 五兵种分类

19个兵种归入5大类，每类内按层级从低到高排列：

| 类别 | 包含兵种 |
|------|---------|
| 步兵 | 民兵 → 步兵 → 枪刺兵 → 斥候小队 → 铁甲兵 |
| 骑兵 | 斥候骑兵 → 护卫骑兵 → 战车 → 突击骑兵 → 重骑兵 |
| 弓兵 | 弓兵 → 弩兵 → 弓骑兵 |
| 攻城 | 冲车 → 投石车 → 弩炮 |
| 水军 | 蒙冲 → 大翼 → 楼船 |

### 8.2 5x5 分类克制表

从基础单位（步兵、弓兵、护卫骑兵、投石车、蒙冲）的已有克制系数推导：

```
攻方↓ 守方→  步兵   骑兵   弓兵   攻城   水军
步兵         1.0    0.8    1.2    0.7    1.0
骑兵         1.3    1.0    0.7    0.6    1.0
弓兵         0.8    1.3    1.0    0.9    1.0
攻城         0.5    0.5    0.8    1.0    0.5
水军         1.0    1.0    1.0    0.5    1.0
```

### 8.3 特殊技能覆盖

- 枪刺兵（`anti_cavalry`）：对骑兵时忽略分类克制表，强制使用1.2倍克制系数

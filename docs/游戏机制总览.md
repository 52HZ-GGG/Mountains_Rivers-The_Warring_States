### 0.4 系统交互总览

```
季节系统 ──→ 资源产出修正 / 民心修正 / 事件条件
    │
    ├──→ 经营系统（粮食/金钱/铁矿产出）
    │       ├──→ 城市经营（建筑效果）
    │       └──→ 粮食与人口（消耗/增长）
    │
    ├──→ 战斗系统（地形修正 / 火攻季节限制）
    │       ├──→ 地理与天堑（地形通过性 / 战斗修正）
    │       └──→ 将军系统（经验/等级）
    │
    ├──→ 民心与稳定性（季节民心修正）
    │       ├──→ 学派系统（基础修正 / 政策触发）
    │       └──→ 文化系统（主流文化安定度惩罚）
    │
    ├──→ 科技树（解锁兵种/建筑/地形通过）
    │
    └──→ 外交系统（好感度/声望/战争状态）
            └──→ 文化传播（游士外交动作）
```

#### 胜利条件

| 路径 | 条件 | 来源 |
|:---|:---|:---|
| 军事胜利 | 征服全部六国首都（6 座） | `balance_params.json` → `victory.military` |
| 文化胜利 | 本国文化覆盖 70% 城市并维持 10 回合 | `balance_params.json` → `victory.cultural` |
| 分数胜利 | 接口预留，未启用 | `balance_params.json` → `victory.score` |

**分数胜利接口**（预留）：
- `enabled: false`：当前未启用
- `target_score: 1000`：目标分数
- `overtime_winner: true`：超时（200 回合）后按分数最高者判定
- `score_weights`：7 项分数来源权重（city_count, city_population, tech_count, military_power, culture_coverage, diplomacy_reputation, wonders_built），全部为 0 待设计

**胜利检查**：每回合结束时 `GameManager.check_victory()` 遍历所有胜利路径 — `game_manager.gd:203`

#### 失败条件

| 条件 | 是否生效 | 来源 |
|:---|:---:|:---|
| 失去所有城市 | ✅ | `balance_params.json` → `defeat.lose_all_cities` |
| 首都陷落且无法迁都 | ✅ | `balance_params.json` → `defeat.capital_lost` + `capital_relocation` |
| 民心归零 | ❌ | `balance_params.json` → `defeat.morale_collapse`（当前否） |
| 超时未胜 | ✅ | `balance_params.json` → `defeat.time_limit_exceeded`（超 200 回合按分数判定） |

**game_over 信号**：`SignalBus.game_over(winner_faction_id: String)` — `signal_bus.gd:21`，触发后游戏进入 `Phase.GAME_OVER` — `game_manager.gd:128-129`

#### 迁都机制

**设计概述**：迁都分两种——被迫迁都（首都陷落后续命）和主动迁都（和平时期迁都）。每座城市有发展度（1~100），影响 AI 迁都选择。

**被迫迁都**（首都陷落触发）：

| 规则 | 说明 |
|:---|:---|
| 触发条件 | 首都被攻陷 |
| 消耗 | 300 金 |
| 稳定期 | 2 回合（新首都安定度 -20，每回合恢复 +10） |
| 次数限制 | 最多 2 次，第 3 次首都陷落判定败北 |
| 民心影响 | 触发被攻陷首都 debuff（民心 -20） |

**主动迁都**（和平时期）：

| 规则 | 说明 |
|:---|:---|
| 触发条件 | 玩家主动发起，当前未处于战争状态 |
| 消耗 | 300 金 |
| 稳定期 | 无（立即生效） |
| 次数限制 | 不计入失败条件判定的迁都次数 |
| 民心影响 | 无 |

**AI 迁都规则**：
- 无迁都次数限制
- 首都陷落时自动选择己方发展度最高的城市
- 消耗 300 金 + 2 回合稳定期（安定度 -20，每回合 +10）

**发展度**（`cities.json` → `development`，1~100）：
- 首都：55~75（咸阳/临淄 75，郢 70，邯郸 70，大梁 65，新郑 60，蓟 55）
- 大城：30~50（蜀郡 35，晋阳 40，彭城 35，洛邑 45 等）
- 中城：15~30
- 小城/边城：5~15
- 中立城：10~45（洛邑 45 为周室旧都，发展度较高）

**攻陷首都效果**：

| 效果 | 参数 | 来源 |
|:---|:---|:---|
| 攻方民心激励 | +5，持续 3 回合（仅首次攻陷某国首都） | `balance_params.json` → `morale.victory_bonus` + `victory_bonus_first_capture_only` |
| 被攻陷国民心 | 立即 -20 | `balance_params.json` → `morale.capital_captured_debuff` |
| 民心恢复 | 每回合 +2，最多恢复到 -10（恢复一半） | `balance_params.json` → `morale.capital_captured_recovery_*` |
| 夺回首都 | 立即清除全部 debuff | `balance_params.json` → `morale.capital_recapture_clear_all` |

**数据来源**：`balance_params.json` → `capital_relocation`（forced/voluntary）、`morale` 段（capital_captured_*）

七国间基于好感度（-100~100）和声望（0~100）的外交博弈。支持战争、停战、同盟、附庸、商路、赠礼、中立城吞并等完整外交行为链。AI 具备独立外交决策能力。

#### 已实现功能

## 5. 民心与稳定性系统

**状态**：⏳ 仅数据定义

#### 设计概述

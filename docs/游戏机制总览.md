#### 迁都机制

**设计概述**：迁都分两种——被迫迁都（首都陷落后续命）和主动迁都（和平时期迁都）。每座城市有发展度（1~100），影响 AI 迁都选择。

**被迫迁都**（首都陷落触发）：

| 规则 | 说明 |
|:---|:---|
| 触发条件 | 首都被攻陷 |
| 消耗 | 300 金 |
| 稳定期 | 2 回合（新首都安定度 -20，每回合恢复 +10） |
| 次数限制 | 最多 2 次，第 3 次首都陷落判定败北 |
| 民心影响 | 触发被攻陷首都 debuff（民心 -20） |

**主动迁都**（和平时期）：

| 规则 | 说明 |
|:---|:---|
| 触发条件 | 玩家主动发起，当前未处于战争状态 |
| 消耗 | 300 金 |
| 稳定期 | 无（立即生效） |
| 次数限制 | 不计入失败条件判定的迁都次数 |
| 民心影响 | 无 |

**AI 迁都规则**：
- 无迁都次数限制
- 首都陷落时自动选择己方发展度最高的城市
- 消耗 300 金 + 2 回合稳定期（安定度 -20，每回合 +10）

**发展度**（`cities.json` → `development`，1~100）：
- 首都：55~75（咸阳/临淄 75，郢 70，邯郸 70，大梁 65，新郑 60，蓟 55）
- 大城：30~50（蜀郡 35，晋阳 40，彭城 35，洛邑 45 等）
- 中城：15~30
- 小城/边城：5~15
- 中立城：10~45（洛邑 45 为周室旧都，发展度较高）

**攻陷首都效果**：

| 效果 | 参数 | 来源 |
|:---|:---|:---|
| 攻方民心激励 | +5，持续 3 回合（仅首次攻陷某国首都） | `balance_params.json` → `morale.victory_bonus` + `victory_bonus_first_capture_only` |
| 被攻陷国民心 | 立即 -20 | `balance_params.json` → `morale.capital_captured_debuff` |
| 民心恢复 | 每回合 +2，最多恢复到 -10（恢复一半） | `balance_params.json` → `morale.capital_captured_recovery_*` |
| 夺回首都 | 立即清除全部 debuff | `balance_params.json` → `morale.capital_recapture_clear_all` |

**数据来源**：`balance_params.json` → `capital_relocation`（forced/voluntary）、`morale` 段（capital_captured_*）

## 8. 粮食·人口·征兵机制

**状态**：⏳ 仅数据定义（决策 88-100 已定稿）

#### 设计概述

以 1 为基本单位的城市人口系统。每人口产出粮食和金钱，自耗粮食，剩余按税率充公至全国储粮。人口增长采用进度条制，受粮食盈余、季节、安定度调节。征兵从总人口中抽取，服役比例过高会惩罚经济和增长。

#### 核心数据模型

**人口产出**（决策 90）：

| 项目 | 基础值 | 修正项 |
|:---|:---:|:---|
| 粮食产出 | 3/人/回合 | ×季节×(1+农田加成)×特产，上限 max_food_production |
| 金钱产出 | 10/人/回合 | ×季节×(1+市集加成)×特产×科技，无上限 |
| 粮食自耗 | 1/人/回合 | 固定 |

**税收**（决策 91）：剩余粮食和金钱按税率（10%-50%，默认30%）充公。税收粮食入全国储粮，税收金钱入全国金钱。

**人口增长**（决策 92）：

```
growth_period = 6.67 / (food_surplus_per_pop / 2.0) / season_mod / stability_mod
```

| 修正项 | 公式 | 值域 |
|:---|:---|:---|
| 粮食盈余 | surplus_per_pop / 2.0 | 盈余2→1.0x，盈余3→1.5x |
| 季节 | 春1.0/夏1.0/秋1.2/冬0.7 | 0.7~1.2 |
| 安定度 | 0.5 + stability/100 | 0.5~1.5（城市级） |

进度条制：每回合推进度 (100/growth_period)%，满100%人口+1。

#### 饥荒与国家储粮不足

**城市饥荒**（当回合城市产粮 < 消耗，决策 93）：
- 该城人口 -1/回合，安定度 -10/回合

**国家储粮不足**（全国储粮 ≤ 0，决策 93）：
- 民心 -5/回合（国家级）
- 军队士气 -10/回合
- 军队攻击 -20%、防御 -20%

#### 征兵系统

**征兵池**（决策 94）：

| 概念 | 定义 |
|:---|:---|
| 最大征召人数 | 全国总人口 × 20% |
| 可服役池 | 全局资源，每回合 +最大征召×10%（开启征兵时） |
| 已服役人数 | 当前军队总人数 |
| 约束 | 可服役 + 已服役 ≤ 最大征召 |

征兵消耗按兵种 cost_gold/cost_food/cost_iron。达到约束上限自动停止。

**征兵衰减**（决策 95）：阵亡 1 人 → 最大征召 -1，每 4 回合恢复 1 点。

**服役惩罚**（决策 96）：服役比例 > 10% 时，每超 1%：粮食产出 -2%、金钱产出 -2%、人口增长 -5%。

**军队维护**（决策 97）：
- 粮食：按兵种 upkeep_food，从全国储粮扣除
- 金钱：统一 5 金/人/回合（暂定）

#### 公式汇总

```
# 城市产出
food_net = population × 3 × season × (1+farm%) × special - population × 1
gold_net = population × 10 × season × (1+market%) × special × tech

# 全国储粮
national_grain += Σ(food_net × tax_rate) - Σ(army_upkeep_food)
national_grain_cap = 200 + Σ(granary_level × 50)

# 人口增长
growth_period = 6.67 / (food_net/population / 2.0) / season_mod / stability_mod
growth_progress += 100 / growth_period

# 征兵
max_cons = total_pop × 0.20
available += max_cons × 0.10 (每回合)
available + active ≤ max_cons

# 服役惩罚
if active/total_pop > 0.10:
    penalty = active/total_pop - 0.10
    food *= (1 - penalty×2), gold *= (1 - penalty×2), growth *= (1 - penalty×5)
```

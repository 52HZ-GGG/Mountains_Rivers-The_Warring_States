## 风险评估

### 阻塞级风险（🔴）
1. **信号未定义（3 处）**：`chain_advanced`/`chain_completed`/`school_exp_gained` 在 event_manager.gd 中被 emit 但 signal_bus.gd 未定义。链式事件推进和学派经验相关事件会在运行时报错。
2. **函数名不匹配**：event_manager.gd 调用 `get_player_faction_id()` 但实际函数名为 `get_player_faction()`。外交效果（好感/声望变化）会在运行时报错。
3. **函数不存在**：event_manager.gd 调用 `GameManager.get_current_school()` 但该函数不存在。含 school 条件的事件触发时会崩溃。
4. **大夫系统缺失**：CombatResolver 预留了大夫字段但从未填充，大夫系统对战斗平衡至关重要。
5. **学派系统不完整**：schools.json 数据丰富但仅战斗加成被使用，学派等级/经验/政策对游戏深度至关重要。

### 重要级风险（🟡）
1. `_has_building()` / `_control_region()` 始终返回 false：影响部分科技解锁条件。
2. `get_power_score()` 硬编码：影响 AI 外交决策精确度。
3. `troops_in_hex_range` 未实现：历史事件（马陵/桂陵之战）永远不会触发。
4. 3 个 JSON 未加载：数据文件已就绪但未接入。

### 建议级风险（🟢）
1. 辖区半径未使用、厌战情绪未实现、互斥科技未检查等为设计增强项，不影响核心游戏循环。

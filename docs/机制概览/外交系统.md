# 外交系统

> 本文档整合《山河策》所有外交系统机制细节，作为程序实现的唯一参考源。
> 所有数值参数从 `data/*.json` 读取，禁止硬编码。
> 本文档基于外交系统重制决策（决策 #65~#85），采用三层外交模型。

---

## 1. 设计概述

七国之间的外交博弈基于三层模型：双边关系层（好感度）、国家声望层（声望）、天子权威层（朝贡度）。支持 18 种外交动作、质子机制、朝贡册封、纵横家事件链、情报力系统，以及俘虏释放等战国特色机制。AI 具有独立的外交决策能力，基于四维性格参数驱动。

情报力系统（含视野与信息可见性机制）详见《情报力系统.md》。

**核心设计原则**：

1. **数据驱动**：所有数值从 `diplomacy.json` / `factions.json` / `balance_params.json` 读取
2. **三层模型**：双边关系 → 国家声望 → 天子权威，三层联动
3. **战国特色**：质子外交、纵横家博弈、朝贡册封、俘虏释放
4. **双向独立**：A 对 B 的好感度 ≠ B 对 A 的好感度
5. **性格驱动 AI**：四维性格（好战/贪婪/诚信/外交）决定 AI 行为

## 2. 三层外交模型

```
┌─────────────────────────────────────────────┐
│            第三层：天子权威层                  │
│         朝贡度（0~100，每国独立）              │
│    朝贡 → 朝贡度↑ → 册封 → 声望↑             │
│    不朝贡 → 朝贡度↓ → 声望↓                   │
└──────────────────┬──────────────────────────┘
                   │ 朝贡度每10点 → 声望+1
┌──────────────────▼──────────────────────────┐
│          第二层：国家声望层                    │
│          声望（0~100，全局单向）               │
│   五档：极低(<20)/低(20-39)/中(40-60)/       │
│         高(61-79)/极高(≥80)                   │
│   效果：贸易/联盟/事件触发                     │
└──────────────────┬──────────────────────────┘
                   │ 声望影响结盟/贸易门槛
┌──────────────────▼──────────────────────────┐
│         第一层：双边关系层                     │
│       好感度（-100~+100，双向独立）            │
│   8种动作 + 俘虏释放 + 质子                   │
└─────────────────────────────────────────────┘
```

## 3. 第一层：双边关系层（好感度）

### 3.1 好感度基础

- **范围**：-100 ~ +100
- **初始值**：来自 `factions.json` 的 `initial_relations` 矩阵
- **双向独立**：A 对 B 和 B 对 A 的好感度独立计算
- **衰减**：每 5 回合向 0 回归 1 点（`opinion_decay.decay_interval_turns` = 5）

**资源交换**：

```
A_to_B = A.all_resource_production × exchange_rate
B_to_A = B.all_resource_production × exchange_rate
// 双方无损，送出不影响自身产出
```

### 3.5 停战谈判

停战谈判采用分数系统，最多 3 轮。

**谈判条件与分数**：

| 条件 | 分数 | 说明 |
|:---|:---|:---|
| 赔款 | 金额×0.01 | 金钱赔偿 |
| 割地 | 每城 2 分 | 割让城市 |
| 称臣 | 3 分 | 成为附庸国 |
| 释放俘虏 | 2 分 | 释放所有被俘大夫 |
| 开放关隘 | 1.5 分 | 指定关隘 N 回合可穿越（不可停留） |
| 互不侵犯 | 1 分 | 停战附带互不侵犯条约 |
| 解除商路封锁 | 0.5 分 | 恢复被切断的商路 |

**AI 接受阈值**：

```
accept_threshold = base_accept_threshold
                 + aggression × personality_mod_aggression
                 + greed × personality_mod_greed
```

- `base_accept_threshold` = 3
- `personality_mod_aggression` = 0.3
- `personality_mod_greed` = 0.2

**AI 停战触发条件**（满足任一即触发）：
- AI 军队损失 > 50%（`ai_troop_loss_threshold`）
- AI 丢失 ≥ 2 座城市（`ai_city_loss_threshold`）
- 战争持续 ≥ 25 回合（`ai_war_weariness_turns`，AI 厌战求和）
- 最短战争回合：5（`min_war_turns`，开战后 5 回合内不触发停战）

### 3.6 俘虏机制与外交整合

俘虏来源：战斗中被俘的武大夫（部队被歼）和文大夫（城破），被俘概率 40%。

**俘虏与外交的关系**：

| 场景 | 效果 |
|:---|:---|
| 拥有对方被俘大夫 | 对方好感度每回合 -1（持续仇恨） |
| 释放俘虏（停战谈判） | 对方好感度 +15，声望 +5 |
| 释放俘虏（独立操作） | 同上 |
| 赎金回收 | 不改变好感度，只回收大夫 |
| 被俘大夫叛变 | 原国好感度 -20 |

**赎金公式**：

```
ransom = tier_base + level × 50
```

| 稀有度 | 基础金 |
|:---|:---|
| 普通 | 100 |
| 稀有 | 250 |
| 传说 | 500 |

| 等级 | 阈值 | 效果 |
|:---|:---|:---|
| 极低 | < 20 | 触发反玩家联盟 + 商路收入 -50% + 结盟成功率 -30% |

**声望等级查询**：

```gdscript
func get_reputation_level(faction_id: String) -> String:
    var rep = get_reputation(faction_id)
    if rep < 20: return "极低"
    if rep < 40: return "低"
    if rep < 61: return "中"
    if rep < 80: return "高"
    return "极高"
```

## 7. 质子机制

### 6.1 核心概念

质子 = 大夫的 HOSTAGE 状态，复用现有大夫系统。任何空闲大夫（武/文/外交）都可以被送作质子。

### 6.2 质子动作

| 动作 | 条件 | 效果 |
|:---|:---|:---|
| 送质子 | 好感度 ≥ 20，双方和平，有空闲大夫 | 好感度 +20，对方不可对你宣战 |
| 召回质子 | 有质子在对方 | 好感度 -15，声望 -5，和平保证解除 |
| 归还质子 | 对方有你的质子 | 好感度 +10 |

数据来源：`diplomacy.json` → `hostage`

### 6.3 质子状态事件

| 状态 | 触发 | 效果 |
|:---|:---|:---|
| 正常 | 送质子成功 | 对方好感度每回合按品质加成（见下表） |
| 逃跑 | 每回合按品质概率 | 好感度 -25，声望 -10，和平保证解除 |
| 死亡 | 每回合按品质概率 | 好感度 -40，声望 -15，和平保证解除，可能引发战争 |
| 自然死亡 | 寿命到期（30 回合） | 无惩罚，仅结束质子状态 |

**品质影响**：

| 品质 | 好感度/回合 | 逃跑概率/回合 | 死亡概率/回合 |
|:---|:---|:---|:---|
| 普通 | +1 | 5% | 2% |
| 稀有 | +2 | 3% | 1% |
| 传说 | +3 | 1% | 0.5% |

数据来源：`diplomacy.json` → `hostage`

### 6.4 质子限制

- 每个国家最多向 1 个国家送质子
- 每个国家最多接收 1 个质子
- 质子不能是已任职的大夫（需为空闲状态）
- 质子期间该大夫不可执行任何行动
- 所有大夫统一 30 回合寿命，寿命到期自然死亡（`balance_params.json` → `minister.lifespan_turns`）

## 8. 纵横家事件系统

### 7.1 概述

纵横家作为独立事件系统，模拟战国时期合纵连横的外交博弈。事件自动触发，影响多国关系。

### 7.2 合纵事件链

| 事件 | 触发条件 | 效果 |
|:---|:---|:---|
| 合纵之议 | 回合 ≥ 10，某国声望 ≥ 70 且正在扩张 | 好感度最低的 3 国自动结盟，对扩张国宣战 |
| 合纵瓦解 | 合纵后 10 回合 | 联盟自动解散，各国好感度 -10 |
| 合纵内讧 | 合纵期间某国获利不均 | 某国退出联盟，好感度 -20 |

### 7.3 连横事件链

| 事件 | 触发条件 | 效果 |
|:---|:---|:---|
| 连横之策 | 回合 ≥ 8，某国声望 ≥ 60 且有强敌 | 好感度最高的 2 国自动结盟，对共同敌人宣战 |
| 连横破裂 | 连横后 8 回合 | 联盟自动解散，各国好感度 -5 |
| 连横反噬 | 连横期间盟友获利过多 | 盟友间好感度 -15 |

### 7.4 纵横家个人事件

| 事件 | 触发条件 | 效果 |
|:---|:---|:---|
| 苏秦出山 | 回合 ≥ 10，齐国存在，齐国声望 ≥ 50 | 齐国获得"合纵"能力 |
| 张仪入秦 | 回合 ≥ 8，秦国存在，秦国声望 ≥ 50 | 秦国获得"连横"能力 |
| 公孙衍游说 | 回合 ≥ 15 | 随机国家好感度变化 ±20 |

#### 合纵能力（vertical_alliance）

一次性消耗能力。发动后选择 1 个目标国 + 指定 2 个盟友国（共 3 国联合），强制临时结盟并对目标国宣战。

| 参数 | 值 | 来源 |
|:---|:---|:---|
| 消耗 | 300 金 + 20 声望 | `strategist_abilities.vertical_alliance` |
| 盟友数量 | 2 个（玩家自选） | `ally_count` |
| 临时同盟持续 | 20 回合 | `forced_alliance_duration` |
| 宣战方式 | 正常宣战（3 回合通知期） | `war_notification: true` |
| 使用次数 | 每局 1 次 | `usage_limit` |

#### 连横能力（horizontal_alliance）

一次性消耗能力。发动后选择 1 个目标国 + 指定 1 个盟友国（共 2 国联合），强制临时结盟并对目标国宣战。

| 参数 | 值 | 来源 |
|:---|:---|:---|
| 消耗 | 200 金 + 15 声望 | `strategist_abilities.horizontal_alliance` |
| 盟友数量 | 1 个（玩家自选） | `ally_count` |
| 临时同盟持续 | 20 回合 | `forced_alliance_duration` |
| 宣战方式 | 正常宣战（3 回合通知期） | `war_notification: true` |
| 使用次数 | 每局 1 次 | `usage_limit` |

**临时同盟规则**：
- 绕过好感度/声望门槛，强制结盟
- 20 回合后自动解散，恢复原外交状态
- 结盟期间所有参与国对目标国处于战争状态
- AI 国家由程序决定是否接受邀请（基于性格参数）

### 7.5 纵横家与三层模型联动

| 层级 | 合纵效果 | 连横效果 |
|:---|:---|:---|
| 双边关系层 | 多国好感度 ±10~30 | 多国好感度 ±10~30 |
| 声望层 | 参与者声望 +5 | 发起者声望 +10 |
| 天子层 | 朝贡度 +5（尊王） | 朝贡度 -5（被视为不义） |

### 7.6 与学派系统联动

- 纵横家学派存在时，合纵/连横事件概率 +50%
- 纵横家学派等级越高，事件效果越强
- 纵横家学派特殊能力：可主动发起合纵/连横

## 9. 国力评分与均势联盟

### 8.1 国力评分公式

```
power_score = city_count × city_weight
            + troop_count × troop_weight
            + tech_count × tech_weight
            + morale × morale_weight
            + culture × culture_weight
```

| 参数 | 权重 | 来源 |
|:---|:---|:---|
| city_weight | 50 | `power_score.city_weight` |
| troop_weight | 3 | `power_score.troop_weight` |
| tech_weight | 15 | `power_score.tech_weight` |
| morale_weight | 1 | `power_score.morale_weight` |
| culture_weight | 1 | `power_score.culture_weight` |

**待补全**：culture 和 tech 当前硬编码为 0，需接入实际值。

### 8.2 均势联盟触发

- **条件**：玩家国力 > 所有 AI 平均国力 × `coalition_trigger_ratio`（1.5）
- **参与者**：好感度 < 0 的 AI 尝试与其他 AI 结盟对抗玩家
- **效果**：声望极低（< 20）时触发反玩家联盟事件

## 15. 跨系统交互

### 15.1 外交 × 经济

- 赠礼/停战赔款消耗金钱
- 商路产生资源交换收入（10% 双向无损）+ 文化扩散加速（×1.5），维护费 10 金/回合
- 朝贡消耗金钱和粮食
- 赎金回收被俘大夫

### 15.2 外交 × 战斗

- 宣战/停战状态决定是否可以攻击
- 军事通行权允许部队穿越他国领土
- 被俘大夫（40% 概率）成为外交筹码
- 背盟攻击力 debuff（×0.9，3 回合）

### 15.3 外交 × 文化

- 文化压制影响好感度
- 游士外交动作（消耗金钱+文化值，对敌方城市文化渗透）
- 商路促进文化交流

### 15.4 外交 × 民心

- 背盟/攻击附庸降低民心
- 僭越称王：选择称王民心 +10，选择维持民心 -10
- 法家学派：无民心惩罚，替换为外交惩罚

### 15.5 外交 × 科技

- 科技"远交近攻"：声望 ≥ 70，外交加成 +0.5
- 科技提供声望加成

### 15.6 外交 × 学派

- 纵横家学派：外交效果加成、联盟费用降低
- 儒家学派：同学派好感度加成
- 法家学派：外交惩罚（-10/-15）
- 墨家学派：强制停战能力

### 15.7 外交 × 大夫

- 外交大夫：降低外交行动费用、被动提升目标好感度
- 大夫技能：巧舌（降低费用）、结交（提升好感度）、离间（降低第三方关系）
- 被俘大夫：成为外交筹码

### 15.8 外交 × 建筑

| 建筑 | 效果 | 状态 |
|:---|:---|:---:|
| 驿站 | 外交费用 -10%/级，情报范围 +1 | ⏳ |
| 王宫 Lv1/2/3 | 声望 +5/10/15 | ⏳ |
| 长城 | 国防 +30%，声望 -5 | ⏳ |

### 15.9 外交 × 情报力

- 好感度是情报力的基础来源（1:1 点数）
- 宣战自动将情报力归零（战争迷雾）
- 突袭宣战停战后 -10 点持续 10 回合
- 商路、质子、外交大夫提供情报力加成
- 驿站/驿馆的情报范围效果通过情报力系统实现

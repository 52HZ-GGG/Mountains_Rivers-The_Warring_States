> 本文档整合《山河策》所有文化系统机制细节，作为程序实现的唯一参考源。
> 所有数值参数从 `data/*.json` 读取，禁止硬编码。

## 1. 设计概述

七国竞争每个城市的文化影响力，文化从城市向外辐射（最大 4 格），按距离衰减。每座城市存储各势力的文化值字典，每回合通过"衰减 + 产出 + 流入"模型结算。任一势力文化占比超过 50% 成为该城主流文化，主流文化与归属国不一致时产生安定度、驻军攻防、民心等多维惩罚。文化覆盖 70% 城市并维持 10 回合可达成文化胜利。

**核心设计原则**：

1. **数据驱动**：所有数值从 `balance_params.json` 读取，禁止硬编码
2. **多途径传播**：6 种传播手段（自然扩散、商路加速、军事占领、学宫输出、奇观加成、游士外交）
3. **七国竞争**：七国共同竞争每个城市的文化值，按比例分配
4. **双刃剑机制**：文化既是胜利路径，也是削弱敌国的战略工具
5. **学派差异化**：六大学派各有独特的文化效果定位

## 2. 文化值体系

### 2.1 文化值存储结构

每座城市存储一个字典，记录各势力在该城市的文化影响力值：

```
city.culture = {
    "qin": 15.0,
    "zhao": 8.0,
    "yan": 3.0,
    "han": 0.0,
    "wei": 0.0,
    "chu": 0.0,
    "qi": 0.0
}
```

- key 为势力 ID（来自 `factions.json`），value 为该势力的文化影响力浮点值
- 所有势力文化值之和不要求等于 100，各势力独立累积
- 新占领城市初始化：占领者文化 = `military_occupation.forced_culture_amount`，其他势力 = 0

### 2.2 文化值完整计算公式

每回合对城市 C 中势力 F 的文化值进行一次性结算，公式如下：

```
culture_new = culture_old × (1 - decay_rate) + production + inflow
```

展开后为：

```
// ===================== 第一步：衰减 =====================
// 辐射输入 = 周边所有城市对城市 C 中势力 F 的扩散量之和
radiation_input = Σ (邻城.culture[F] × spread_rate × distance_decay × season_mul × school_mul)

// 动态衰减率：无辐射 30%，满辐射 20%，始终有衰减防止文化值膨胀
decay_rate = base_decay_rate - decay_reduction × radiation_input
decay_rate = clamp(decay_rate, base_decay_rate - decay_reduction, base_decay_rate)

// 道家特殊：己方文化衰减率减少（Lv3 -50%，值来自 schools.json → level_effects.self_culture_decay_reduction）
if city.owner.school == "daoism":
    decay_rate = decay_rate × (1 - self_culture_decay_reduction)

culture_after_decay = culture_old × (1 - decay_rate)

// ===================== 第二步：产出 =====================
// 基础产出 = 城市等级产出 + 学宫产出
culture_base = city_level × base_culture_per_level + academy_production

// 百分比加成（加法叠加）
culture_buff = school_culture_spread_rate              // 学派文化传播率加成（儒家 Lv1 +15%/Lv2 +30%/Lv3 +50%）
             + tech_bonus                             // 科技加成（私学之兴 +10% 等）
             + wonder_bonus                           // 奇观加成（章华台 +30% 等）
             + special_resource_bonus                 // 特产加成（+20%）

// 文化特产城市：学宫产出额外 +50%
if city.has_special_resource("culture"):
    academy_production = academy_production × (1 + academy_effect_bonus)

production = culture_base × (1 + culture_buff)

// ===================== 第三步：流入 =====================
// 对每个邻城 N（距离 ≤ 4）：
for each neighbor N:
    spread = N.culture[F] × spread_rate × distance_decay[N→C]

    // 季节修正
    spread = spread × season_multiplier                // 冬季 ×0.5，其他 ×1.0

    // 学派修正（扩散方，值来自 schools.json → level_effects.culture_spread_rate）
    spread = spread × school_spread_multiplier         // 儒家 Lv1 ×1.15 / Lv2 ×1.30 / Lv3 ×1.50

    // 商路修正
    if has_trade_route(N, C):
        spread = spread × trade_route_multiplier       // ×1.5

    // 接收方防御修正（值来自 schools.json → level_effects，按等级递进叠加）
    if C.owner.school == "legalism":                   // 法家：外来抵抗 Lv2 30%/Lv3 50%
        spread = spread × (1 - foreign_culture_resistance)
    if C.owner.school == "mohism":                     // 墨家：文化防御 Lv1 15%/Lv2 30%/Lv3 50%
        spread = spread × (1 - culture_defense_bonus)
    if C.owner.school == "daoism":                     // 道家：外来传播 Lv2 -25%/Lv3 -40%
        spread = spread × (1 - culture_incoming_reduction)

    // 关隘阻断
    if 关隘在 N→C 路径上 and 关隘非 C.owner 占领:
        spread = 0

    inflow += spread

// ===================== 第四步：学派特殊效果 =====================
// 法家：每回合净化已有外来文化（Lv3 -8点/回合，低于阈值30停止净化）
// 值来自 schools.json → level_effects.foreign_culture_suppression_per_turn
// 阈值来自 balance_params.json → culture.foreign_culture_suppression_cap
if C.owner.school == "legalism" and F != C.owner:
    if culture_after_decay > foreign_culture_suppression_cap:
        culture_after_decay = culture_after_decay - foreign_culture_suppression_per_turn
        culture_after_decay = max(culture_after_decay, foreign_culture_suppression_cap)

// ===================== 最终结果 =====================
culture_new = culture_after_decay + production + inflow
```

**结算顺序**：衰减 → 产出 → 法家压制 → 流入，四步在同一回合内依次执行。

**示例**：赵国 3 级城市，Lv2 学宫，儒家，拥有百家争鸣 + 稷下学宫，无特产。齐国 2 级邻城（距离 1）对赵国文化值 20，有商路。

```
// 衰减（假设赵国在该城文化值 50，齐国辐射输入 1.95，clamp 至 1.0）
radiation_input = 20 × 0.05 × 1.0 × 1.0 × 1.3 × 1.5 = 1.95   // 齐国辐射（含儒家+商路）
radiation_input = min(radiation_input, 1.0)                       // clamp 至最大值
decay_rate = 0.3 - 0.1 × 1.0 = 0.2                               // 满辐射，最低衰减 20%
culture_after_decay = 50 × (1 - 0.2) = 40

// 产出
culture_base = 15 + 25 = 40
culture_buff = 0.3 + 0.3 + 0.2 = 0.8
production = 40 × 1.8 = 72

// 流入（齐国对赵国城市的扩散）
spread = 20 × 0.05 × 1.0 × 1.0 × 1.3 × 1.5 = 1.95

// 最终
culture_new = 40 + 72 + 1.95 = 113.95
```

### 2.3 衰减率速查

| 辐射输入 | 衰减率 | 含义 |
|:---:|:---:|:---|
| 0（无辐射） | 30%/回合 | 无文化源支撑，快速衰减 |
| 0.25 | 27.5%/回合 | 弱辐射 |
| 0.5 | 25%/回合 | 中等辐射 |
| 0.75 | 22.5%/回合 | 强辐射 |
| 1.0（满辐射） | 20%/回合 | 强力辐射源，最低衰减 |

- 始终存在 20%~30% 的衰减，防止文化值无限膨胀
- 文化胜利需要持续产出和辐射维持，难度适中

## 5. 主流文化判定

### 5.1 判定规则

当某势力文化在城市中的占比超过 50% 时，该势力成为该城的主流文化（决策 #68）：

```
total_culture = sum(city.culture[faction] for all factions)
for each faction:
    ratio = city.culture[faction] / total_culture
    if ratio > flip_threshold:
        mainstream_culture = faction
        break
else:
    mainstream_culture = null    // 无主流文化
```

| 参数 | 值 | 来源 |
|:---|:---:|:---|
| 主流文化阈值 | 50% | `balance_params.json` → `culture.flip_threshold` |

### 5.2 文化占比计算

```
culture_ratio[F] = city.culture[F] / sum(city.culture[faction] for all factions)
```

- 所有势力文化占比之和 = 100%
- 如果所有势力文化值均为 0，则无主流文化

## 6. 文化不匹配惩罚

**不一致定义**：城市主流文化为**外国文化**（其他势力的文化占比 > 50%）。如果城市不存在主流文化（没有任何文化占比 > 50%），则**不触发**任何惩罚。

### 6.1 安定度惩罚

| 效果 | 值 | 来源 |
|:---|:---:|:---|
| 安定度惩罚 | -15 | `balance_params.json` → `stability.culture_mismatch_penalty` |

### 6.2 驻守部队攻防惩罚

| 效果 | 值 | 来源 |
|:---|:---:|:---|
| 驻守部队攻击 | -20%（加法层） | `balance_params.json` → `culture.culture_mismatch_garrison_atk_penalty` |
| 驻守部队防御 | -20%（加法层） | `balance_params.json` → `culture.culture_mismatch_garrison_def_penalty` |

- 驻守部队攻防 -20% 在加法层叠加，即与其他百分比 buff（如首都驻军 +20%、季节修正等）相加后统一乘算

### 6.3 首都文化不匹配额外惩罚

当**首都**主流文化不一致时，额外触发：

| 效果 | 值 | 来源 |
|:---|:---:|:---|
| 全国民心下降 | -10（固定值） | `balance_params.json` → `culture.culture_mismatch_capital_morale_penalty` |

- 只要首都主流文化不一致即持续生效

## 7. 文化翻转

### 7.1 翻转阈值

当非归属势力的文化在城市中占比超过 50% 时，城市可能叛离至该文化所属势力。

| 参数 | 值 | 来源 |
|:---|:---:|:---|
| 翻转阈值 | 50% | `balance_params.json` → `culture.flip_threshold` |

### 7.2 翻转效果

- 翻转后城市归属变更，建筑保留
- 翻转判定每回合结算时进行
- 翻转后该城市中其他势力文化值清零，新归属势力文化值保留

### 8.2 资源交换机制

商路使两城的所有资源产出进行交换：

```
// 城市 A 与城市 B 的商路
A_to_B = A.all_resource_production × exchange_rate    // A 的产出送给 B
B_to_A = B.all_resource_production × exchange_rate    // B 的产出送给 A
// 双方无损，送出不影响自身产出
```

| 参数 | 值 | 来源 |
|:---|:---:|:---|
| 资源交换比例 | 10% | `balance_params.json` → `culture.trade_route.resource_exchange_rate` |

- "所有资源产出"包括：金钱、粮食、木材、铁、马匹、工匠、建材等
- 交换是双向且无损的——送出方不减少自身产出

### 8.3 文化传播加速

| 效果 | 值 | 来源 |

- 与其他加成（学派、科技等）乘算

### 8.4 商路持续与破坏

## 9. 文化传播途径

### 9.1 邻城自然扩散

文化从高浓度城市向低浓度邻城扩散，最大范围 4 格，按距离衰减。

- 每回合自动计算
- 详见第 4 节

### 9.2 贸易路线文化加速

有商路的城市间，文化扩散速率 ×1.5。

- 双向生效
- 详见第 8.3 节

### 9.3 军事占领强制传播

攻占城市时，立即注入占领者文化并削弱原归属势力文化：

| 效果 | 值 | 来源 |
|:---|:---:|:---|
| 注入己方文化 | +50 | `balance_params.json` → `culture.military_occupation.forced_culture_amount` |
| 削弱原归属势力文化 | -50 | `balance_params.json` → `culture.military_occupation.culture_reduction_amount` |

- 一次性效果，不改变后续扩散率
- 与 `stability.city_captured_penalty` (-30) 配合，占领造成安定度和文化的双重冲击

### 9.4 学宫建筑主动输出

学宫（academy）建筑提供城市文化产出，产出的文化值自然向外辐射。

- Lv1 产出 +10，Lv2 产出 +25
- 全国限建 2 座
- 详见第 3.2 节

### 9.5 奇观全国加成

奇观提供全国范围的文化产出加成：

| 奇观 | 效果 | 来源 |
|:---|:---|:---|
| 章华台 | 全国文化产出 +30% | `wonders.json` → `zhanghuatai.effects.culture_production_national` |
| 稷下学宫 | 全国文化产出 +20% | `wonders.json` → `jixia_academy.effects.culture_production_national` |

### 9.6 游士外交动作

游士是主动文化传播手段，代表战国时期的游说之士（苏秦、张仪、孟子等）。

| 规则 | 值 | 来源 |
|:---|:---|:---|
| 金钱消耗 | 100 金 | `balance_params.json` → `culture.scholar.cost_gold` |
| 效果量 | ±30 点 | `balance_params.json` → `culture.scholar.culture_effect_amount` |
| 冷却时间 | 5 回合 | `balance_params.json` → `culture.scholar.cooldown_turns` |
| 目标限制 | 任意城市 | `balance_params.json` → `culture.scholar.target_restriction` |

**操作方式**：二选一

| 操作 | 效果 | 说明 |
|:---|:---|:---|
| 增加己方文化 | 目标城市己方文化 +30 | 文化渗透 |
| 削弱敌方文化 | 目标城市指定势力文化 -30 | 文化压制 |

- 对称设计：两种操作消耗和效果量相同
- 冷却是全局的（每 5 回合可用一次，非按目标）
- 无科技前置条件

**AI 使用逻辑**：

| 条件 | 行为 | 来源 |
|:---|:---|:---|
| 目标城市敌方文化占比 > 30% | 考虑使用游士 | `balance_params.json` → `culture.scholar.ai_use_threshold` |
| 冷却随机偏移 | +0~2 回合 | `balance_params.json` → `culture.scholar.ai_cooldown_randomness` |

- AI 优先选择：己方文化 30~45% 的城市（接近主流阈值）、最强对手的城市、己方领土邻近的城市

## 10. 学派文化效果

### 10.1 儒家——文化扩散主导

儒家以仁德教化百姓，文化传播能力最强。

| 效果 | 值 | 来源 |
|:---|:---:|:---|
| 文化传播率 Lv1 | +15% | `schools.json` → `level_effects.culture_spread_rate` |
| 文化传播率 Lv2 | +30%（叠加后） | `schools.json` → `level_effects.culture_spread_rate` |
| 文化传播率 Lv3 | +50%（叠加后） | `schools.json` → `level_effects.culture_spread_rate` |
| 春秋季节文化传播 Lv2 | +10% | `schools.json` → `level_effects` 季节联动 |
| 春秋季节文化传播 Lv3 | +15% | `schools.json` → `level_effects` 季节联动 |

- 战略定位：文化胜利首选学派，被动扩散能力最强
- Lv3 春秋两季总加成：+65%（50% + 15%），夏冬 +50%

### 10.2 法家——文化净化

法家以严刑峻法治国，主动净化外来文化影响（决策 #115）。

| 效果 | 值 | 来源 |
|:---|:---:|:---|
| 外来文化传播抵抗 Lv2 | 30% | `schools.json` → `level_effects.foreign_culture_resistance` |
| 外来文化传播抵抗 Lv3 | 50%（叠加后） | `schools.json` → `level_effects.foreign_culture_resistance` |
| 每回合净化外来文化 Lv3 | -8 点 | `schools.json` → `level_effects.foreign_culture_suppression_per_turn` |
| 净化停止阈值 | 30 | `balance_params.json` → `culture.foreign_culture_suppression_cap` |

- 外来文化传播抵抗：其他势力对己方城市的文化扩散量 ×(1 - resistance)
- 文化净化机制：每回合主动清除己方城市中所有非己方文化值 8 点，但当外来文化值已低于 30 时停止净化
- 净化停止阈值防止法家将外来文化完全清零，保留最低限度的文化多样性

```
// 法家文化净化伪代码
if C.owner.school == "legalism" and F != C.owner:
    if culture_after_decay > foreign_culture_suppression_cap:  // 阈值 30
        culture_after_decay = culture_after_decay - foreign_culture_suppression_per_turn  // -8
        culture_after_decay = max(culture_after_decay, foreign_culture_suppression_cap)    // 不低于 30
```

- 战略定位：最佳文化防御学派，主动净化能力独一无二

### 10.3 墨家——文化入侵抵抗

墨家主张兼爱非攻，擅长防御。

| 效果 | 值 | 来源 |
|:---|:---:|:---|
| 文化防御加成 Lv1 | +15% | `schools.json` → `level_effects.culture_defense_bonus` |
| 文化防御加成 Lv2 | +30%（叠加后） | `schools.json` → `level_effects.culture_defense_bonus` |
| 文化防御加成 Lv3 | +50%（叠加后） | `schools.json` → `level_effects.culture_defense_bonus` |

- 所有外来文化对己方城市的扩散量减少对应百分比
- 与法家的抵抗机制不同：墨家是通用防御，法家是特定抵抗
- 战略定位：文化防御为辅，军事防御为主

### 10.4 道家——文化惯性

道家主张无为而治，文化具有"惯性"——既抵抗外来影响，又不易衰减。

| 效果 | 值 | 来源 |
|:---|:---:|:---|
| 外来文化传播减少 Lv2 | 25% | `schools.json` → `level_effects.culture_incoming_reduction` |
| 外来文化传播减少 Lv3 | 40%（叠加后） | `schools.json` → `level_effects.culture_incoming_reduction` |
| 己方文化自然衰减减少 Lv3 | 50% | `schools.json` → `level_effects.self_culture_decay_reduction` |

- 外来传播减少：其他势力对己方城市的扩散量 ×(1 - reduction)
- 衰减减少：己方文化值的衰减率 ×(1 - 0.50)（如原衰减 20% → 道家为 10%）
- 战略定位：己方文化一旦建立就持久存在，适合长期经营

### 10.5 兵家——征服文化传播

兵家以奇制胜，征服即传播。

| 效果 | 值 | 来源 |
|:---|:---:|:---|
| 占领强制文化传播加成 Lv3 | +50% | `schools.json` → `level_effects.occupation_culture_bonus` |
| 征服文化传播范围 Lv3 | +2 格 | `schools.json` → `level_effects.conquest_culture_spread_range` |

- 占领加成：军事占领的文化注入量从 50 提升至 75（50 × 1.5）
- 征服传播范围：占领城市时，文化注入同时影响被占城市周围 2 格内的所有己方城市（注入量 50% 强度，即 37.5 点）
- 战略定位：军事扩张 = 文化扩张，闪电战式文化策略

### 10.6 纵横家——文化外交加成

纵横家以口舌为剑，擅长利用外交和贸易进行文化渗透。

| 效果 | 值 | 来源 |
|:---|:---:|:---|
| 贸易路线文化加成 Lv1 | +20% | `schools.json` → `level_effects.trade_route_culture_bonus` |
| 贸易路线文化加成 Lv2 | +40%（叠加后） | `schools.json` → `level_effects.trade_route_culture_bonus` |
| 贸易路线文化加成 Lv3 | +60%（叠加后） | `schools.json` → `level_effects.trade_route_culture_bonus` |
| 游士消耗减少 Lv2 | 20% | `schools.json` → `level_effects.scholar_cost_reduction` |
| 游士消耗减少 Lv3 | 40%（叠加后） | `schools.json` → `level_effects.scholar_cost_reduction` |
| 文化压制好感惩罚减弱 Lv3 | 5 | `schools.json` → `level_effects.diplomacy_culture_opinion_bonus` |

- 贸易路线文化加成：商路的文化扩散倍率额外提升
- 游士消耗减少：降低游士行动成本
- 文化压制好感惩罚减弱：己方文化压制别国时，减弱目标对己方的好感降低效果 5 点（正值 = 减少负面效果）
- 战略定位：通过贸易和游士进行文化渗透，纵横家可减轻文化压制带来的外交代价

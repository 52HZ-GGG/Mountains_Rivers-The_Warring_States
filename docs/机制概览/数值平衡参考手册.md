### 2.4 战斗推算

> **战斗公式**: `伤害 = max((攻击 × 攻击加成 × 克制系数) × COEFF / (COEFF + 防御 × 防御加成), 1) × 随机系数(0.9~1.1)`
>
> **COEFF = 20**（防御系数），防御值越高减伤比例越大但永远不为 0

**公式说明**:
- **攻击加成**: 地形修正 + 士气修正 + 科技修正 + 势力修正 + 学派修正 + 伏击/火攻加成
- **防御加成**: 地形修正 + 科技修正 + 势力修正 + 学派修正 + 建筑修正
- **克制系数**: 根据兵种类型矩阵查询（步兵×骑兵=1.3等）
- **乘法模型**: 防御不再直接扣减攻击，而是按比例减伤（COEFF=20 时：5防减伤20%，10防减伤33%，18防减伤47%）
- **保底伤害**: 最低为1，确保不会出现0伤害

#### 2.4.4.1 防御城市建筑数值

**核心概念说明**:
- **结构HP（structure_hp）**: 防御建筑本身的HP，攻击时会先消耗建筑HP
- **防御加成（defense_bonus）**: 城墙提供的百分比防御加成，应用于城市整体防御
- **城攻击加成（city_attack_bonus）**: 瓮城提供的城市攻击百分比加成
- **塔攻击（tower_attack）**: 箭塔每回合对攻城军队的攻击伤害

**攻城伤害分配机制**:
```
攻城伤害 = 攻击方总攻击 × 攻城加成 × 随机系数
├── 50% → 驻军HP（damage_split_garrison = 0.5）
└── 50% → 城墙结构HP（damage_split_wall = 0.5）
```
